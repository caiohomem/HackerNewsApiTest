# HackerNewsApiTest
Senior Backend Developer Coding Test

Como executar:

Clonar a solução e executar com o Visual Studio 2019 e com o .Net Core 2.2 instalado na máquina.

O browser abre com a interface do Swagger.

Clicar no botões "GET", "Try It Out" e "Execute" nesta ordem.

O resultado é exibido na caixa de texto "Response Body"

Foi criado um memory cache para melhorar a performance da aplicação
